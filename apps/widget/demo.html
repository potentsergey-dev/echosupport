EchoSupport Widget Demo

Виджет должен появиться в правом нижнем углу.
По умолчанию demo использует Docker stack на http://localhost:8080. Передайте реальный ключ агента через query string: ?agentKey=pk_...&apiBase=http://localhost:8080.

Чтобы получить ключ: откройте /admin, выберите агента, перейдите на вкладку Embed и скопируйте Public agent key. Добавьте origin demo http://localhost:5173 в разрешённые источники на вкладке Профиль.
